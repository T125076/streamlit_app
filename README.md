# 📊 消費者物価指数と家計消費支出の関係分析（最終課題）

本アプリは、政府統計ポータルサイト e-Stat に掲載されているデータを用いて、  
**消費者物価指数（CPI）** と **世帯の消費支出** の関係を可視化・分析することを目的とした最終課題である。  

物価の変動が家計の消費行動にどのような影響を与えているかを、  
グラフ表示および回帰分析によって確認する。

---

## ■ 課題の目的

e-Stat に公開されている統計データから複数のデータを選択し、  
それらを整理・可視化することで社会・経済の動向を理解する。  

本課題では、物価水準を示す消費者物価指数と、  
家計の実際の消費行動を表す消費支出に着目し、  
両者の関係性を分析することを目的とする。

---

## ■ 使用データ

- 消費者物価指数（年度別）
- 世帯の消費支出（年度別）

いずれも e-Stat（政府統計ポータルサイト）より取得したデータを使用している。  
取得したデータは Excel ファイル `支出.xlsx` に整理して読み込んでいる。

---

## ■ システム概要

本アプリは Python と Streamlit を用いて Web アプリケーションとして実装した。  

主な機能は以下の通りである。

- CPI と消費支出の推移を折れ線グラフで表示（双軸表示）
- CPI と消費支出の関係を散布図と回帰線で表示
- 表示年度範囲の変更
- 相関係数および回帰式の自動計算
- データ表の表示切替

---

## ■ 分析方法

消費者物価指数（CPI）は家計が購入する商品・サービスの価格変動を示す指標であるため、  
税や貯蓄を含む総支出ではなく、実際の消費行動を表す**消費支出**を用いて分析を行った。  

また、指数と支出は桁が大きく異なるため、  
折れ線グラフでは双軸表示を採用し、視認性の向上を図っている。  

さらに、散布図では回帰直線を描画し、  
CPI と消費支出の関係を定量的に評価した。
